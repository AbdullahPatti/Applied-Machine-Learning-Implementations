# Machine Learning Foundations

A structured repository dedicated to implementing core machine learning algorithms from first principles and validating them against standard library implementations.  

This repository focuses on mathematical understanding, implementation clarity, and empirical evaluation rather than black-box usage of high-level APIs.

---

## Purpose

The objective of this repository is to:

- Implement classical machine learning algorithms manually
- Understand their statistical assumptions
- Analyze optimization behavior
- Compare results with scikit-learn
- Evaluate performance through structured experiments

The emphasis is on **depth, not volume**.

---

## What This Repository Contains

This repository will include implementations of:

- Probabilistic classifiers
- Linear models
- Multiclass strategies (OVR, OVO)
- Optimization techniques
- Performance evaluation methods

Each module includes:

- Conceptual explanation
- Mathematical formulation
- Clean implementation
- Experimental results
- Benchmark comparison with scikit-learn
- Observations and limitations

## Evaluation Approach

All models are evaluated using:

- Accuracy
- Precision
- Recall
- F1-score
- Confusion matrix
- Class-wise analysis

Where relevant, comparisons are made against scikit-learn to validate correctness and performance.

---

## Design Principles

- No unnecessary abstractions
- Transparent mathematical logic
- Explicit training loops
- Stable numerical computation
- Clear experimental documentation

---

## Tech Stack

- Python
